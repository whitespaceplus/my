留白＋ WHITESPACE｜Multi-page static site (KF-like structure)

Files:
- index.html (home)
- projects.html
- pricing.html
- blog.html
- contact.html
- style.css
- main.js

How to use:
1) 解壓縮後，雙擊 index.html 即可預覽。
2) 上線：把整個資料夾內容上傳到 GitHub Pages / Netlify / 任意靜態主機。

Notes:
- Projects/Testimonials/Clients/Stats 都是「可替換內容的佔位」。
- Contact 表單目前是示意版本；若要可提交，可接 Formspree/Netlify Forms。
